# Product Sales Dashboard
## Overview

This project presents an interactive Product Sales Dashboard built in Microsoft Excel to analyze business performance across products, regions, customer types, and sales teams. 
The dashboard transforms raw sales data into actionable insights through dynamic visualizations and key performance indicators.

## Features
  - Interactive slicers for filtering by year, region, customer type, and region manager
  - KPI cards highlighting total revenue, average order value, top-performing region, and return rate
  - Visual analysis of product revenue, discounts, salespersons, and store performance
  - Comparative insights across multiple years
  - Business-focused recommendations based on findings

### Key Insights
  - Total Revenue: $4.7M+
  - Average Order Value: $3,149.83
  - Top Region by Sales: North
  - Return Rate: 24.78%

### Tools Used
  - Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - KPI Cards
  - Data Cleaning & Transformation

### Project Objectives
  - Evaluate overall sales performance
  - Identify top-performing products, regions, and salespersons
  - Analyze discount and return trends
  - Support strategic business decisions with data insights
### Business Value
This dashboard enables stakeholders to monitor performance, identify opportunities for growth, and address operational challenges such as high return rates and uneven regional performance.
